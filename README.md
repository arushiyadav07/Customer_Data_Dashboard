# Customer_Data_Dashboard
Power BI Dashboard for Customer Segmentation Analysis using Excel data.
## Project Overview
This project is created using Microsoft Power BI to analyze customer data based on age, gender, category, purchase amount, and visit frequency.

## Features
- Customer Purchase Analysis
- Gender Distribution
- Category-wise Purchase
- Age-wise Purchase Trend
- Interactive Filters

## Tools Used
- Microsoft Power BI
- Microsoft Excel

## Files
- Customer_data_Dashboard.pbix
- Customer_data.xlsx
